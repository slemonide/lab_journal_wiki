Abstract

  • Non-uniform, deterministic sampling patterns
  • Introduce a computable parameter, which is a function of the sampling pattern
  • If this parameter is small, then we may recover missing entries of the matrix, with appropriate weights
  • Recovery method: project the (zero-padded) subsampled matrix onto the set of low-rank matrices
  • Under non-uniform deterministic sampling, this method yields a biased solution
  • Algorithm to de-bias it
  • De-biasing significantly improves the estimate
  • With noise, errors of this method can be sub-optimal when sampling is highly non-uniform
  • To remedy this, project onto the max-norm ball whose robustness to noise tolerates arbitrary non-uniform sampling
  • Analysis of convex optimization in this framework

Introduction

Previous work:

  • Work of Heiman et al. seems interesting. They talk about expander graphs. In particular, they show that if algorithm works well on a non-uniform random sampling pattern, then there exists a deterministic sampling pattern on which it also does well.
  • Lee and Shraibman introduce a parameter which measures the complexity of the sampling pattern.
  • This parameter is given by solving a semidefinite program involving the sampling pattern.
  • If that parameter is small, then the entries of the matrix can be recovered with appropriate weights.
  • Their results do not work with noisy observations and do not apply to projection-based methods.

This work:

  • Continue in the same vein as Lee and Shraibman, by identifying a parameter of the sampling pattern that guarantees good weighted recovery.
  • Parameter is a special case of Lee and Shraibman.
  • More specific => can say more
  • This parameter is easy to compute
  • Generalization of the spectral gap approach in the uniform case
  • If this parameter is small, then the missing entries of the matrix can be estimated well, even in the presence of noise.
  • Analyze several efficient algorithms for actually doing this

More precisely:

  • Projection-based methods for matrix completion are biased under such non-unifrom sampling (what about uniform?).
  • Method to de-bias them
  • Algorithm to reconstruct a matrix given deterministic non-uniform sampling patterns.
  • Another method based on convex optimization
  • Numerical simulations for a projection-based method using real-data sampling patterns.
  • These show that debiasing is vital for accurate matrix recovery.

Possible future work: adapt this for the case of subspace estimation for one-iteration ALS algorithm? Seems to go well in hand with the topic of this paper.

Computable parameter

  • Matrix completion vs low-rank matrix recovery?
  • Parameters which determine that the problem set up is well conditioned are generally NP-hard to compute.
  • Restricted isometry constant of compressive sensing is NP-hard to verify.
  • Restricted isometry constants defined in the context of low-rank matrix recovery.
  • Guarantees that can be made using parameters that are known to be computable in polynomial time is generally far from optimal.

Spectral gap:

  • If the spectral gap of the graph is large (first to eigenvalues), then the matrix can be efficiently estimated with near-optimal guarantees.
  • These results require that the top singular vectors of Ω {\displaystyle \Omega } be the all-ones vectors; in particular, the best-rank one approximation must be equal to all-ones matrix (uniform case).
  • This work extends this to settings where this may not be the case (i.e. any practical problem).

Consider two extreme cases:

  • Best rank-1 approximate to Ω {\displaystyle \Omega } is proportional to 1 {\displaystyle 1} . So, can hope to recover all the entries of M {\displaystyle M} .
  • Ω {\displaystyle \Omega } only contains a single row of ones, and the rest is zero. Thus, it is itself rank-1 , and is its own best rank-1 approximation. In this case, it is clear that all we can recover is M ∘ Ω {\displaystyle M\circ \Omega } , and nothing else.

This motivates the following weighted approach:

  • Suppose that W {\displaystyle W} is the best rank-1 approximation to Ω {\displaystyle \Omega } ; we will attempt to recover W ∘ M {\displaystyle W\circ M} .

To be precise:

  • Let W {\displaystyle W} be any rank-1 matrix, so that every entry of W {\displaystyle W} is strictly greater than 0.
  • Theory works whether or not W {\displaystyle W} is the best rank-1 approximation of Ω {\displaystyle \Omega } .
  • Let λ := | | Ω − W | | {\displaystyle \lambda :=||\Omega -W||} . I.e. this is sort of like standard deviation.
  • λ {\displaystyle \lambda } (with some assumptions) is the second singular value of W {\displaystyle W} .
  • If λ {\displaystyle \lambda } is small, then a low-rank matrix can be well-approximated just from viewing entries on Ω {\displaystyle \Omega } .

Structure of W {\displaystyle W} is important:

  • If it is not flat, the projection method of Keshevan, Montanari and Oh gives a biased estimate.
  • Show how to debias the estimate.
  • Error bounds are weighted proportionally to W {\displaystyle W} .
  • One expects to have more accurate estimate of rows/columns that are sampled more often
