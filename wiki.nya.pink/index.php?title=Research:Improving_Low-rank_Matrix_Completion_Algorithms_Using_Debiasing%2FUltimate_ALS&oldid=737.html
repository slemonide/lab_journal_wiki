Rank=1

Let's first solve rank=1 case.

In this case, second equation is easy: r 1 2 + r 2 2 + ⋯ + r n 2 = 1 {\displaystyle r_{1}^{2}+r_{2}^{2}+\cdots +r_{n}^{2}=1}

First equation is harder, but still can be expressed in an easy-to-understand form. First, let's compute R R T {\displaystyle RR^{T}} :

R R T = [ r 1 2 r 1 r 2 ⋯ r 1 r n r 2 r 1 r 2 2 ⋯ r 2 r n ⋮ ⋮ ⋱ ⋮ r n r 1 r n r 2 ⋯ r n 2 ] {\displaystyle RR^{T}={\begin{bmatrix}{\color {red}r_{1}^{2}}&{\color {red}r_{1}r_{2}}&{\color {red}\cdots }&{\color {red}r_{1}r_{n}}\\r_{2}r_{1}&{\color {red}r_{2}^{2}}&{\color {red}\cdots }&{\color {red}r_{2}r_{n}}\\\vdots &\vdots &{\color {red}\ddots }&{\color {red}\vdots }\\r_{n}r_{1}&r_{n}r_{2}&\cdots &{\color {red}r_{n}^{2}}\end{bmatrix}}}

Idea: If we have a quadratic system of n {\displaystyle n} unknowns, we can express it as a linear system of equations of n ( n − 1 ) {\displaystyle n(n-1)} unknowns, where each unknown would correspond to a product of some of these two unknowns. We then solve it to get values for products of all n ( n − 1 ) {\displaystyle n(n-1)} unknowns. After that, we can just take the squared terms, and take square roots of their values to get the result!

Overall, system above gives us n 2 + 1 {\displaystyle n^{2}+1} equations in n 2 − n {\displaystyle n^{2}-n} linear unknowns. If we had higher rank, we would have had n 2 r 2 − n r {\displaystyle n^{2}r^{2}-nr} unknowns. For r {\displaystyle r} large enough, the system above would be under determined. Actually, in rank>1 case, we would have a system of n 2 + r 2 {\displaystyle n^{2}+r^{2}} equations.

Question: would linearized system give identity as an answer?

Answer: No. The orthonormality property r 1 2 + r 2 2 + ⋯ + r n 2 = 1 {\displaystyle r_{1}^{2}+r_{2}^{2}+\cdots +r_{n}^{2}=1} will not work if we were to set all diagonal terms to ones.

Note: Actually, we can probably get away with having to use only n ( n + 1 ) 2 {\displaystyle {\frac {n(n+1)}{2}}} unknown linear variables. In summary, we have:

  • system of n 2 + r 2 {\displaystyle n^{2}+r^{2}} equations
  • n ( n + 1 ) 2 {\displaystyle {\frac {n(n+1)}{2}}} unknown linear variables

I believe we should also be able to fix r 2 {\displaystyle r^{2}} variables for the vector by multiplying it by an arbitrary invertible r × r {\displaystyle r\times r} matrix:

  • n ( n + 1 ) 2 − r 2 {\displaystyle {\frac {n(n+1)}{2}}-r^{2}} unknown linear variables

Note: I believe that multiplying rank=r matrices by each other gives rank=r matrices again. Hence, the first equation actually corresponds to a system of about 2 n r {\displaystyle 2nr} equations, which is probably not that bad.

After solving the system of linear equations, we then would get a number for each of those. Next, we can just fix r {\displaystyle r} of those to prevent rotations, and then iteratively compute the remaining ones values by padding to construct invertible matrices (hopefully they will be invertible!).

Equation counting

Initially, we have a system of n + 1 {\displaystyle n+1} homogeneous quadratic equations in n {\displaystyle n} unknowns.

We then linearize this system by treating each product as a variable, r i r j = a i j {\displaystyle r_{i}r_{j}=a_{ij}} , we have n + 1 {\displaystyle n+1} linear equations in n ( n + 1 ) 2 {\displaystyle {\frac {n(n+1)}{2}}} unknowns.

We solve it so that we can express n + 1 {\displaystyle n+1} variables in terms of ( n + 1 ) ( n − 2 ) 2 {\displaystyle {\frac {(n+1)(n-2)}{2}}} variables. So, if n = 2 {\displaystyle n=2} , we are done.

We are free to choose these n + 1 {\displaystyle n+1} variables. A natural choice seems to be: r i 2 {\displaystyle r_{i}^{2}} , and r 1 r 2 {\displaystyle r_{1}r_{2}} .

So, we express these variables in terms of other variables, r i r j {\displaystyle r_{i}r_{j}} , where i ≠ j {\displaystyle i\not =j} , and where we don't have r 1 r 2 {\displaystyle r_{1}r_{2}} .

The idea now is to multiply the known variables into all reasonable combinations of unknown variables, and their squares.

Say we have an equation of the form a = b x + d y {\displaystyle a=bx+dy} . We would square it to get a 2 = b 2 x 2 + d 2 y 2 + b d x y {\displaystyle a^{2}=b^{2}x^{2}+d^{2}y^{2}+bdxy}

For example, we would multiply r 1 2 {\displaystyle r_{1}^{2}} and r 3 2 {\displaystyle r_{3}^{2}} because right hand side of that product would have r 1 r 3 {\displaystyle r_{1}r_{3}} and r 1 2 r 3 2 {\displaystyle r_{1}^{2}r_{3}^{2}} . Such operation would also eliminate monomials r 1 2 {\displaystyle r_{1}^{2}} and r 2 2 {\displaystyle r_{2}^{2}} .

Now, let's count how many combinations we can get this way.

In terms of number of linear variables, we get p + p + p ( p − 1 ) 2 = p ( p + 3 ) 2 {\displaystyle p+p+{\frac {p(p-1)}{2}}={\frac {p(p+3)}{2}}} equations.

In terms of original variables, this will be ( n + 1 ) ( n − 2 ) ( n 2 − n + 4 ) 8 {\displaystyle {\frac {(n+1)(n-2)(n^{2}-n+4)}{8}}} new variables.

In terms of equations, we would have

From the r i 2 r j 2 {\displaystyle r_{i}^{2}r_{j}^{2}} terms, we get ( n − 1 ) ( n − 2 ) 2 {\displaystyle {\frac {(n-1)(n-2)}{2}}} combinations.

We now also multiply the r 1 r 2 {\displaystyle r_{1}r_{2}} by all the unknown terms r i r j {\displaystyle r_{i}r_{j}} , but where i = 1 {\displaystyle i=1} , or j = 2 {\displaystyle j=2} . This allows us to use the following identities:

  • ( r 1 r 2 ) ( r 1 r j ) = ( r 1 2 ) ( r 2 r j ) {\displaystyle (r_{1}r_{2})(r_{1}r_{j})=(r_{1}^{2})(r_{2}r_{j})}
  • ( r 1 r 2 ) ( r i r 2 ) = ( r 2 2 ) ( r 1 r i ) {\displaystyle (r_{1}r_{2})(r_{i}r_{2})=(r_{2}^{2})(r_{1}r_{i})}

We can then replace ( r 1 r 2 ) {\displaystyle (r_{1}r_{2})} , ( r 1 2 ) {\displaystyle (r_{1}^{2})} , and ( r 2 2 ) {\displaystyle (r_{2}^{2})} by some expressions written in terms of unknown variables. This will give us additional 2 ( n − 1 ) {\displaystyle 2(n-1)} variables.

Recursion

Notice that products r i 2 r j 2 {\displaystyle r_{i}^{2}r_{j}^{2}} and ( r 1 r 2 ) ( r i r j ) {\displaystyle (r_{1}r_{2})(r_{i}r_{j})} are made up from the following terms:

  • r i r j {\displaystyle r_{i}r_{j}}
  • ( r i r j ) 2 {\displaystyle (r_{i}r_{j})^{2}}
  • ( r i r j ) ( r k r l ) {\displaystyle (r_{i}r_{j})(r_{k}r_{l})}

And those terms are products of expressions that we already know. So, we can actually keep expanding these to generate even more equations. We can generate as many equations as we want!
