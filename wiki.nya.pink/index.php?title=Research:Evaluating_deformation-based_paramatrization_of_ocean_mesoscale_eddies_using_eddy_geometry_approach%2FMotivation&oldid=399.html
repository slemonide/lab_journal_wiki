The gist of this project is to parametrize the effect of small-scale eddies (which might not be resolvable) on the large-scale deformation.

More detailed:

  • We are testing Anstey Zanna 2017 deformation based parametrization of partially-resolved eddies with regards to eddy geometry framework.

Model

The model used is of a jet that is similar to western boundary jet and Kuroshiro extension.

It is two dimensional, and hence uses streamfunction (which is a scalar two dimensional field) to describe the flow of liquid. This is possible due to the fact that liquid is not compressible (up to some degree).

We denote it by:

ψ = ψ ( x , y ) {\displaystyle \psi =\psi (x,y)} .

To generate model output, run the Fortran script (TODO: add link?). This will produce a matrix where each entry corresponds to an average value of the streamfunction in some region (10x10 km?).

Hence, when computing derivatives, we have to use numerical schemes (and have to feed in the right physical sizes of entries so that units make sense).

Getting other things from the streamfunction

As is described in the Wikipedia article, velocity can be written in terms of streamfunction as:

u = ∂ ψ ∂ y , v = − ∂ ψ ∂ x {\displaystyle u={\frac {\partial \psi }{\partial y}},\qquad v=-{\frac {\partial \psi }{\partial x}}}

Where u {\displaystyle u} and v {\displaystyle v} are the flow velocity components in the cartesian x {\displaystyle x} and y {\displaystyle y} coordinate directions, respectively. TODO: there are special names for these coordinate directions in the context of geophysical fluid dynamics.

Now, as we have seen from the videos, there are two other things that we might be interested in: vorticity and deformation.

Vorticity:

ζ = ∂ v ∂ x − ∂ u ∂ y {\displaystyle \zeta ={\frac {\partial v}{\partial x}}-{\frac {\partial u}{\partial y}}}

Deformation:

δ = D 2 + D ~ 2 {\displaystyle \delta ={\sqrt {D^{2}+{\widetilde {D}}^{2}}}}

Where D {\displaystyle D} is stretch:

D = ∂ u ∂ y + ∂ v ∂ x {\displaystyle D={\frac {\partial u}{\partial y}}+{\frac {\partial v}{\partial x}}}

And D ~ {\displaystyle {\widetilde {D}}} is shear:

D ~ = ∂ u ∂ x − ∂ v ∂ y {\displaystyle {\widetilde {D}}={\frac {\partial u}{\partial x}}-{\frac {\partial v}{\partial y}}}

Eddy Anisotropy

Now, a more complicated concept is of eddy anisotropy. It comes from eddy geometry framework (TODO: add a link for eddy geometry intro).

It is defined as:

L = ( u ′ u ′ ¯ − v ′ v ′ ¯ ) 2 4 + u ′ v ′ ¯ 2 {\displaystyle {\color {blue}L={\sqrt {{\frac {({\overline {u'u'}}-{\overline {v'v'}})^{2}}{4}}+{\overline {u'v'}}^{2}}}}}

Parametrization

The main result is that

L = κ | ζ | δ {\displaystyle {\color {blue}L}={\color {red}\kappa |\zeta |\delta }}

where κ {\displaystyle \kappa } and partitioning scale are position dependent.

Dummy streamfunctions

Should we try to test the parametrization on some dummy streamfunctions? For example, on

ψ = 1 2 ( x 2 + y 2 ) = 1 2 r 2 {\displaystyle \psi ={\frac {1}{2}}(x^{2}+y^{2})={\frac {1}{2}}r^{2}}

In this case

u = ∂ ψ ∂ y = y and v = − ∂ ψ ∂ x = − x {\displaystyle u={\frac {\partial \psi }{\partial y}}=y\quad {\text{and}}\quad v=-{\frac {\partial \psi }{\partial x}}=-x}

Thus ζ = ∂ v ∂ x − ∂ u ∂ y = − 1 − 1 = − 2 , D = ∂ u ∂ y + ∂ v ∂ x = 1 − 1 = 0 ,  and  D ~ = ∂ u ∂ x − ∂ v ∂ y = 0 − 0 = 0 {\displaystyle \zeta ={\frac {\partial v}{\partial x}}-{\frac {\partial u}{\partial y}}=-1-1=-2,D={\frac {\partial u}{\partial y}}+{\frac {\partial v}{\partial x}}=1-1=0,{\text{ and }}{\widetilde {D}}={\frac {\partial u}{\partial x}}-{\frac {\partial v}{\partial y}}=0-0=0}

Hence δ = 0 {\displaystyle \delta =0}

From L = κ | ζ | δ {\displaystyle {\color {blue}L}={\color {red}\kappa |\zeta |\delta }} this leads to the fact that L = ( u ′ u ′ ¯ − v ′ v ′ ¯ ) 2 4 + u ′ v ′ ¯ 2 = 0 {\displaystyle {\color {blue}L={\sqrt {{\frac {({\overline {u'u'}}-{\overline {v'v'}})^{2}}{4}}+{\overline {u'v'}}^{2}}}}=0}

And thus u ′ v ′ ¯ = 0  and  u ′ u ′ ¯ = v ′ v ′ ¯ {\displaystyle {\overline {u'v'}}=0{\text{ and }}{\overline {u'u'}}={\overline {v'v'}}}

Parametrization applicability

Is parametrization applicable to static example of fields? Like the ones studied in example streamfunctions? Or is it more applicable to dynamic fields that actually change in time and have some specific statistical properties because of that?

Like, it is true that one can also study spacial variability without even thinking about time variability at all.

Meaning of means

To continue we need to understand what the means u ′ v ′ ¯ , u ′ u ′ ¯ ,  and  v ′ v ′ ¯ {\displaystyle {\overline {u'v'}},{\overline {u'u'}},{\text{ and }}{\overline {v'v'}}} mean.

Let's consider them as spatial means. We can then formally define mean of some field f ( x , y ) = f ( p ) {\displaystyle f(x,y)=f(p)} at point ( x 0 , y 0 ) = p 0 {\displaystyle (x_{0},y_{0})=p_{0}} as

f ( p 0 ) ¯ = ∫ f ( p ) σ ( p 0 , p ) d p {\displaystyle {\overline {f(p_{0})}}=\int f(p)\sigma (p_{0},p)dp}

Here σ ( p 0 , p ) {\displaystyle \sigma (p_{0},p)} is a density function with the following properties:

  1. ∫ σ ( p 0 , p ) d p = 1 {\displaystyle \int \sigma (p_{0},p)dp=1}
  2. σ ( p 0 , p ) ≥ 0 {\displaystyle \sigma (p_{0},p)\geq 0}
  3. p 0 ∈ argmax p σ ( p 0 , p ) {\displaystyle p_{0}\in {\text{argmax}}_{p}\sigma (p_{0},p)}
  4. lim | | p | | − > inf ( p + p 0 ) = 0 {\displaystyle \lim _{||p||->\inf }(p+p_{0})=0}
  5. It should be possible to specify the length scale of the features that mean captures. I.e. Dirac delta function is not a practical density function here.

The first two properties come from the fact that we want an average (perhaps a weighted one). The last two properties come from physical considerations: average at some point is mostly influenced by a value at that point and not at all by points very far away.

Notice that for each point p 0 {\displaystyle p_{0}} we essentially have a different function of p {\displaystyle p} , σ p 0 ( p ) = σ ( p 0 , p ) {\displaystyle \sigma _{p_{0}}(p)=\sigma (p_{0},p)} .

Large scale flow

To be honest, we also should compute ζ {\displaystyle \zeta } and δ {\displaystyle \delta } from large-scale field. It makes sense that there is a connection between definition of the large-scale field and the way average is computed for velocity.

Thus, we define large-scale fields:

u ( p 0 ) ¯ = ∫ u ( p ) σ ( p 0 , p ) d p and v ( p 0 ) ¯ = ∫ v ( p ) σ ( p 0 , p ) d p {\displaystyle {\overline {u(p_{0})}}=\int u(p)\sigma (p_{0},p)dp\quad {\text{and}}\quad {\overline {v(p_{0})}}=\int v(p)\sigma (p_{0},p)dp}

Scale-invariant vortex

For our example above:

u ( x 0 , y 0 ) ¯ = ∫ y σ ( x 0 , y 0 | x , y ) d x d y and v ( x 0 , y 0 ) ¯ = ∫ − x σ ( x 0 , y 0 | x , y ) d x d y {\displaystyle {\overline {u(x_{0},y_{0})}}=\int y\sigma (x_{0},y_{0}|x,y)dxdy\quad {\text{and}}\quad {\overline {v(x_{0},y_{0})}}=\int -x\sigma (x_{0},y_{0}|x,y)dxdy}

This example is very simple and corresponds to a vortex flow. So, on the large scale it should look the same as on the small scale. That is, it should be scale-invariant.

Hence, for this example:

u ( x 0 , y 0 ) ¯ = u ( x 0 , y 0 ) and v ( x 0 , y 0 ) ¯ = v ( x 0 , y 0 ) {\displaystyle {\overline {u(x_{0},y_{0})}}=u(x_{0},y_{0})\quad {\text{and}}\quad {\overline {v(x_{0},y_{0})}}=v(x_{0},y_{0})}

From this we get:

u ( x 0 , y 0 ) = ∫ y σ ( x 0 , y 0 | x , y ) d x d y and v ( x 0 , y 0 ) = ∫ − x σ ( x 0 , y 0 | x , y ) d x d y {\displaystyle u(x_{0},y_{0})=\int y\sigma (x_{0},y_{0}|x,y)dxdy\quad {\text{and}}\quad v(x_{0},y_{0})=\int -x\sigma (x_{0},y_{0}|x,y)dxdy}

Notice that we expect that u ( x 0 , y 0 ) = y 0 {\displaystyle u(x_{0},y_{0})=y_{0}} and v ( x 0 , y 0 ) = − x 0 {\displaystyle v(x_{0},y_{0})=-x_{0}} .

We split σ ( x 0 , y 0 | x , y ) {\displaystyle \sigma (x_{0},y_{0}|x,y)} as σ ( x 0 , y 0 | x , y ) = σ X ( x 0 | x ) σ Y ( y 0 | y ) {\displaystyle \sigma (x_{0},y_{0}|x,y)=\sigma _{X}(x_{0}|x)\sigma _{Y}(y_{0}|y)} so that ∫ x σ X ( x 0 | x ) d x = ∫ y σ Y ( y 0 | y ) d y = 1 {\displaystyle \int x\sigma _{X}(x_{0}|x)dx=\int y\sigma _{Y}(y_{0}|y)dy=1}

This gives us:

u ( x 0 , y 0 ) = ∫ y σ Y ( y 0 | y ) d y and v ( x 0 , y 0 ) = ∫ − x σ X ( x 0 | x ) d x {\displaystyle u(x_{0},y_{0})=\int y\sigma _{Y}(y_{0}|y)dy\quad {\text{and}}\quad v(x_{0},y_{0})=\int -x\sigma _{X}(x_{0}|x)dx}

To make this exact we could use Dirac delta functions. But then again, we want to be able to have some notion of scale (see property 5 of σ {\displaystyle \sigma } TODO: add an actual reference that captures some semantics). So, instead we could choose a Gaussian function σ X ( x 0 | x ) = 1 a 2 π e − 1 2 ( x − x 0 a ) 2 {\displaystyle \sigma _{X}(x_{0}|x)={\frac {1}{a{\sqrt {2\pi }}}}e^{-{\frac {1}{2}}\left({\frac {x-x_{0}}{a}}\right)^{2}}} and the same for σ Y ( y 0 | y ) {\displaystyle \sigma _{Y}(y_{0}|y)} . Here a {\displaystyle a} denotes the scale.

Note: we can actually also choose a simple uniform probability function centered at the point of interest. This could be more useful for applications. Here is the function:

σ X 1 ( x 0 | x ) = { 1 a for  x ∈ [ x 0 − a 2 , x 0 + a 2 ] 0 otherwise {\displaystyle \sigma _{X}^{1}(x_{0}|x)={\begin{cases}{\frac {1}{a}}&{\text{for }}x\in [x_{0}-{\frac {a}{2}},x_{0}+{\frac {a}{2}}]\\0&{\text{otherwise}}\end{cases}}}

Now, v ( x 0 , y 0 ) = ∫ − x σ X ( x 0 | x ) d x = 1 a 2 π ∫ − x e − 1 2 ( x − x 0 a ) 2 d x = − x 0 {\displaystyle v(x_{0},y_{0})=\int -x\sigma _{X}(x_{0}|x)dx={\frac {1}{a{\sqrt {2\pi }}}}\int -xe^{-{\frac {1}{2}}\left({\frac {x-x_{0}}{a}}\right)^{2}}dx=-x_{0}}

And, for the other function, v ( x 0 , y 0 ) = ∫ − x σ X 1 ( x 0 | x ) d x = 1 a ∫ x 0 − a 2 x 0 + a 2 − x d x = − x 0 {\displaystyle v(x_{0},y_{0})=\int -x\sigma _{X}^{1}(x_{0}|x)dx={\frac {1}{a}}\int _{x_{0}-{\frac {a}{2}}}^{x_{0}+{\frac {a}{2}}}-xdx=-x_{0}}

This is a very interesting result: on every scale, the flow looks identical. This is because it is scale-invariant.

We also see that this happens because the Gaussian function is symmetric, and the change in velocity is also symmetric. So a natural question arises: Can we always find a scaling function (i.e. density distribution), so that for any flow, it leaves the flow unchanged at all scales?

General properties of density function for scale-invariant vortex

General properties are:

  1. x 0 = ∫ x σ ( x 0 | x ) d x {\displaystyle x_{0}=\int x\sigma (x_{0}|x)dx}
  2. 1 = ∫ x ∂ ∂ x 0 σ ( x 0 | x ) d x {\displaystyle 1=\int x{\frac {\partial }{\partial x_{0}}}\sigma (x_{0}|x)dx}
  3. 0 = ∫ x ∂ 2 ∂ x 0 2 σ ( x 0 | x ) d x {\displaystyle 0=\int x{\frac {\partial ^{2}}{\partial x_{0}^{2}}}\sigma (x_{0}|x)dx}

Gaussian flow

As a second example of a valid streamfunction, consider:

ψ = exp ⁡ ( − r 2 2 λ 2 ) = exp ⁡ ( − x 2 + y 2 2 λ 2 ) {\displaystyle \psi =\exp(-{\frac {r^{2}}{2\lambda ^{2}}})=\exp(-{\frac {x^{2}+y^{2}}{2\lambda ^{2}}})}

This streamfunction is notable because it is not the same on all scales and has a very specific scale of λ {\displaystyle \lambda } .
