Abstract

Introduction

  • V {\displaystyle V} : the set of nodes of a point-to-point communication network.
  • G = ( V , E ) {\displaystyle G=(V,E)} : a directed graph representing the network.
    • ( i , j ) ∈ E ⟺ {\displaystyle (i,j)\in E\iff } information can be sent noiselessly from node i {\displaystyle i} to node j {\displaystyle j}
  • X 1 , … , X m {\displaystyle X_{1},\dots ,X_{m}} : mutually independent information sources
  • h i {\displaystyle h_{i}} : information rate of X i {\displaystyle X_{i}} , in bits per unit time
    • h = [ h 1 ⋯ h m ] {\displaystyle \mathbf {h} =[h_{1}\cdots h_{m}]}
  • a : { 1 , … , m } → V , b : { 1 , … , m } → 2 V {\displaystyle a:\{1,\dots ,m\}\to V,b:\{1,\dots ,m\}\to 2^{V}} : mappings
    • Source X i {\displaystyle X_{i}} is generated at node a ( i ) {\displaystyle a(i)} and multicast to all nodes in b ( i ) {\displaystyle b(i)} .

Together a , b , h {\displaystyle a,b,\mathbf {h} } completely describe a set of multicast requirements.

Let R i j ≥ 0 {\displaystyle R_{ij}\geq 0} be a real number associated with the edge ( i , j ) {\displaystyle (i,j)} ("capacity").

Let R = [ R i j , ( i , j ) ∈ E ] {\displaystyle \mathbf {R} =[R_{ij},(i,j)\in E]} . (How are the entries of this vector arranged? Does it matter?)

R {\displaystyle \mathbf {R} } is admissible if and only if there is a coding scheme satisfying the multicast requirements ( a , b , h ) {\displaystyle (a,b,\mathbf {h} )} so that the coding rate from node i {\displaystyle i} to node j {\displaystyle j} is less than or equal to R i j {\displaystyle R_{ij}} for all edges.

Goal: To characterize the admissible coding rate region R {\displaystyle {\mathcal {R}}} ; th set of all admissible R {\displaystyle \mathbf {R} } for any graph G {\displaystyle G} and requirements ( a , b , h ) {\displaystyle (a,b,\mathbf {h} )} .

A max-flow min-cut theorem

This theorem characterizes the admissible coding rate region for the single source problem, that is where

{ a ( 1 ) = s b ( 1 ) = { t 1 , ⋯ , t L } {\displaystyle {\begin{cases}a(1)=s\\b(1)=\{t_{1},\cdots ,t_{L}\}\end{cases}}}

Information source X 1 {\displaystyle X_{1}} is generated at node s {\displaystyle s} (the source) and multicast to nodes t ! , … , t L {\displaystyle t_{!},\dots ,t_{L}} (the sinks). This is the one-source L {\displaystyle L} -sink problem.

New notation:

  • The subgraph of G {\displaystyle G} from s {\displaystyle s} to t ℓ {\displaystyle t_{\ell }} is the graph G ℓ = ( V , E ℓ ) {\displaystyle G_{\ell }=(V,E_{\ell })} where
E ℓ = { ( i , j ) ∈ E : ( i , j )  is on a directed path from  s  to  t ℓ } {\displaystyle E_{\ell }=\{(i,j)\in E:(i,j){\text{ is on a directed path from }}s{\text{ to }}t_{\ell }\}} .
  • F = [ F i j , ( i , j ) ∈ E ] {\displaystyle \mathbf {F} =[F_{ij},(i,j)\in E]} is a flow in G {\displaystyle G} from s {\displaystyle s} to t ℓ {\displaystyle t_{\ell }} if for all ( i , j ) ∈ E , 0 ≤ F i j ≤ R i j {\displaystyle (i,j)\in E,0\leq F_{ij}\leq R_{ij}} such that for all i ∈ V {\displaystyle i\in V} except for s {\displaystyle s} and t ℓ {\displaystyle t_{\ell }} , total flow into node i {\displaystyle i} is equal to the total flow out of node i {\displaystyle i} .
  • F i j {\displaystyle F_{ij}} is the value of F {\displaystyle \mathbf {F} } in edge ( i , j ) {\displaystyle (i,j)} .
  • The value of F {\displaystyle \mathbf {F} } is defined as
∑ j : ( s , j ) ∈ E F s j − ∑ i : ( i , s ) ∈ E F i s = ∑ i : ( i , t ℓ ) ∈ E F i t ℓ − ∑ j : ( t ℓ , j ) ∈ E F t ℓ j {\displaystyle \sum _{j:(s,j)\in E}F_{sj}-\sum _{i:(i,s)\in E}F_{is}=\sum _{i:(i,t_{\ell })\in E}F_{it_{\ell }}-\sum _{j:(t_{\ell },j)\in E}F_{t_{\ell }j}}
  • F {\displaystyle F} is a max-flow from s {\displaystyle s} to t ℓ {\displaystyle t_{\ell }} if its value is ≥ {\displaystyle \geq } that of any other flow from s {\displaystyle s} to t ℓ {\displaystyle t_{\ell }} .
Conjecture 1. ( R , h , G ) {\displaystyle (\mathbf {R} ,h,G)} is admissible if and only if the values of a max-flow from s {\displaystyle s} to t ℓ {\displaystyle t_{\ell }} for any ℓ {\displaystyle \ell } are greater than or equal to h {\displaystyle h} , the rate of the information source.

This is similar to the max-flow min-cut theorem in graph theory.

In computer science and optimization theory, the max-flow min-cut theorem states that in a network flow, the maximum amount of flow passing from the source to the sink is equal to the total weight of the edges in the minimum cut, i.e. the smallest total weight of the edges which if removed would disconnect the source from the sink.
