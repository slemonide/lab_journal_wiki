Preliminaries

Hermite was the first to have considered quadratic binary forms with conjugate indeterminates of the type: f ( , y ) = a x x − b x y 0 − b 0 x 0 y + c y y 0 {\displaystyle f(,y)=axx_{-}bxy_{0}-b_{0}x_{0}y+c_{y}y_{0}} where a , c ∈ R {\displaystyle a,c\in \mathbb {R} } , b {\displaystyle b} and b 0 {\displaystyle b_{0}} are complex conjugates; these constants are not assumed to be integers [TRANSLATE]. x {\displaystyle x} and y {\displaystyle y} are complex integers of the type α + β i {\displaystyle \alpha +\beta i} , where α , β {\displaystyle \alpha ,\beta } are real integers, and x 0 , y 0 {\displaystyle x_{0},y_{0}} are their complex conjugates.

The real-valued invariant δ = a c − b b 0 {\displaystyle \delta =ac-bb_{0}} plays an important role:

  • If δ > 0 {\displaystyle \delta >0} , the form is definite; since one can assume a , c {\displaystyle a,c} to be positive, the form is positive for all values of x {\displaystyle x} and y {\displaystyle y} .
  • If δ < 0 {\displaystyle \delta <0} , the form is indefinite; depending on x , y {\displaystyle x,y} it may assume values of either sign.
  • If δ = 0 {\displaystyle \delta =0} , the form decomposes into the product of two conjugate linear forms ( λ x + μ y ) ( λ 0 x 0 + μ 0 y 0 ) {\displaystyle (\lambda x+\mu y)(\lambda _{0}x_{0}+\mu _{0}y_{0})} . These forms do not have proper quadratic character and so will be ignored (?) [TRANSLATE]

This chapter is only about quadratic forms, so let's skip to the next chapter which considers forms of arbitrary degree. We will focus on the cubic case.

Note: We can think of δ = a c − b b 0 {\displaystyle \delta =ac-bb_{0}} as a determinant of [ a b b 0 c ] {\displaystyle {\begin{bmatrix}a&b\\b_{0}&c\end{bmatrix}}}

And, as we know, determinants are invariant to unitary transformations (i.e. "rotations"). Another such invariant is trace. Also eigenvalues (but not eigenvectors). In general, all scalar, direction-independent properties of matrices are invariants.

Theory of binary forms with complex coefficients and variables. Reduction of non-quadratic forms

Consider the form

f ( x , y ) = a 0 x n + a 1 x n − 1 y + ⋯ + a n − 1 x y n − 1 + a n y n {\displaystyle f(x,y)=a_{0}x^{n}+a_{1}x^{n-1}y+\cdots +a_{n-1}xy^{n-1}+a_{n}y^{n}}

where a 0 , a 1 , … , a n {\displaystyle a_{0},a_{1},\dots ,a_{n}} are arbitrary complex numbers, and the homogenous equation

f ( z , 1 ) = a 0 z n + a 1 z n − 1 + ⋯ + z n = 0 {\displaystyle f(z,1)=a_{0}z^{n}+a_{1}z^{n-1}+\cdots +z_{n}=0}

whose roots are called the roots or zeroes of the form f {\displaystyle f} . Supposing that a 0 ≠ 0 {\displaystyle a_{0}\neq 0} we can write

f ( x , y ) = a 0 ( x − z 1 y ) ( x − z 2 y ) ⋯ ( x − z n y ) {\displaystyle f(x,y)=a_{0}(x-z_{1}y)(x-z_{2}y)\cdots (x-z_{n}y)}

denoting the roots of f {\displaystyle f} by z 1 , z 2 , … , z n {\displaystyle z_{1},z_{2},\dots ,z_{n}} , and if a 0 , a 1 , … , a p − 1 {\displaystyle a_{0},a_{1},\dots ,a_{p-1}} are zero while a p ≠ 0 {\displaystyle a_{p}\neq 0} , then

f ( x , y ) = a p y p ( x − z p + 1 y ) ⋯ ( x − z n y ) {\displaystyle f(x,y)=a_{p}y^{p}(x-z_{p+1}y)\cdots (x-z_{n}y)}

and z p + 1 , … , z n {\displaystyle z_{p+1},\dots ,z_{n}} are the finite roots (?) of f {\displaystyle f} .

Now consider that with the real quantities t 1 2 , t 2 2 , … , t n 2 {\displaystyle t_{1}^{2},t_{2}^{2},\dots ,t_{n}^{2}} we can construct the positive definite quadratic form with conjugate indeterminates as follows:

ϕ ( x , y ) = t 1 2 ℜ ( x − z 1 y ) + t 2 2 ℜ ( x − z 2 y ) + ⋯ + t n 2 ℜ ( x − z n y ) {\displaystyle \phi (x,y)=t_{1}^{2}\Re (x-z_{1}y)+t_{2}^{2}\Re (x-z_{2}y)+\cdots +t_{n}^{2}\Re (x-z_{n}y)} [what is this fraktur symbol? read earlier parts to figure out. I think it's the real part]

Now we imagine that the t {\displaystyle t} 's are assigned determinate values and for this system of values we reduce the form ϕ {\displaystyle \phi } . To do this we perform the substitution

{ x = α X + β Y y = γ X + δ Y {\displaystyle {\begin{cases}x=\alpha X+\beta Y\\y=\gamma X+\delta Y\end{cases}}}

where α δ − β γ = 1 {\displaystyle \alpha \delta -\beta \gamma =1} . In fact this is just the group action of ( α β γ δ ) ∈ S L ( 2 , C ) {\displaystyle {\begin{pmatrix}\alpha &\beta \\\gamma &\delta \end{pmatrix}}\in SL(2,\mathbb {C} )} .

Then imagine that for any possible values of t {\displaystyle t} , for each of these systems we reduce the form ϕ {\displaystyle \phi } and that we make the same substitution in f {\displaystyle f} that reduces ϕ {\displaystyle \phi } [TRANSLATE]. This way, one will obtain a class of equivalent forms to f {\displaystyle f} that we will denote ( f ) {\displaystyle (f)} and whose properties we will study.

It all comes back to studying the collection of substitutions (S) that must be done to reduce ϕ {\displaystyle \phi } for all values of t {\displaystyle t} . This is facilitated by the representation of definite quadratic forms with conjugate variables and the geometric interpretation of these forms, which were recalled in the introduction.

First of all, we must find which domain describes the representative point of ϕ {\displaystyle \phi } when values of t {\displaystyle t} vary.
